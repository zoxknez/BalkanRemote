# 🚀 START HERE - RemoteBalkan Scraper

## ⚡ Quick Start (3 koraka)

```bash
# 1. Setup
cd scraper
pip install -r requirements.txt

# 2. Scrape + Load
python scrape.py --remote --limit 200 && python scrape.py --load

# 3. Check
python scrape.py --check
```

✅ **GOTOVO!** Fresh jobs u bazi.
